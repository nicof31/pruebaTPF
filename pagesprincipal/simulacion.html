Ejemplo nivel: AVANZADO

Gestor de tareas para taller

Probar simulador
Detalle de funcionamiento

Simulador realizado utilizando en el código ingreso de datos en el input y creación de objetos en formato JSON y verificación con localStorage, se utilizan objetos, arrays, ciclo for, los datos generados se muestran en consola y en el DOM con creación de tabla con los datos ingresados, con generación de ID de tarea automático. Se realiza proceso del array/objetos con cálculos matemáticos para procesar la información y calcular el precio de trabajo en la tarea y cálculo que nos entregue información total de ganancias generadas y cantidad de tareas cargadas. Tiene la particularidad de contar con dos botones uno para guarde los datos de cada tarea ingresada en el inpunt y otro botón que borrar todos los datos guardados en localStorage y recarga automáticamente la pagina para que se refresque la información. La información generada de las tareas se guarda en localStorage la lista, para se convierte el JSON a formato texto para que el localStorage se pueda cargar, después de agregar la tarea se llama la función para consumir los datos del localStorage.
